JavaScript : 世界上被误解最多的编程语言

-- Douglas Crockford

关于名字 - JavaScript

JavaScript 不是 Java ！

JavaScript 是一门解释性脚本语言
与 C 相比，它牺牲了性能，以提升动态特性和开发效率

JavaScript 被定型

它只能在Web浏览器里跑

  • 在服务端：Node.js
  • 在桌面客户端：Windows 8

JavaScript 有设计上的失误

没有一门编程语言是完美的

  • + 号被同时用作数字的加法，以及字符串的拼接
  • with 语句
  • ; 自动插入

JavaScript 是面向对象的？

它连 class 都没有！
